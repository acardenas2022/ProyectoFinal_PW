El proyecto consiste en un sistema de servicio postal.
Se desarrolla significativamente el backend y el sitio WEB, y se comienza a desarrollar el frontend.
El backend y el sitio WEB se construyeron con PHP y Mysql para todo lo que es el funcionamiento.
La interaccion del usuario se creo a traves del framework Materialize (https://materializecss.com/), siendo adaptado al proyecto. 
El sistema corre bien con PHP 8.1.13, MySQL 5.7.33, Apache 2.4.47

Es compatible con php 8.1.13 en adelante. 
Es compatible con MySQL 5.0 en adelante.
